# Repeat

> Fonte: [Repeat](https://dwarffortresswiki.org/index.php/Repeat) — Dwarf Fortress Wiki (GFDL/MIT)

**Repeating** a task at a workshop is a method of automating mass production. By using the menu and selecting a task, the key can be used to toggle the repeating of a task. Tasks currently set to repeat are marked with a blue **R**. If the same workshop has multiple tasks marked as repeat, those tasks will be cycled.

Check your needs from time to time - you don't want to end up with, say, a gazillion rock doors in your stockpiles.
