# Init.txt

> Fonte: [Init.txt](https://dwarffortresswiki.org/index.php/Init.txt) — Dwarf Fortress Wiki (GFDL/MIT)

*Many of these settings are covered in detail here.*

The **init.txt** file can be used to make many adjustments to the game presentation and feel, like changing screen size, or enabling tilesets. Other options that would improve framerate by deactivating features are contained in d_init.txt.

This file is located at ```/prefs/init.txt` along with the embark profiles, d_init.txt, and other useful files.
