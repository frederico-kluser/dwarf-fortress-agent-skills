# Ranger

> Fonte: [Ranger](https://dwarffortresswiki.org/index.php/Ranger) — Dwarf Fortress Wiki (GFDL/MIT)

**Ranger** is a category profession for dwarves that work with live animals, both creatures and vermin. Rangers hunt, train, and care for animals.

An adult dwarf whose best skill is in this category and whose total experience within this category is distributed across multiple skills will be known as a Ranger - if the dwarf's experience is mostly concentrated within a single skill, the dwarf will be known for that particular skill instead. Graphically, rangers will often wear dark green tunics.
