# Places

> Fonte: [Places](https://dwarffortresswiki.org/index.php/Places) — Dwarf Fortress Wiki (GFDL/MIT)

The **place information menu** is accessed from the main view by clicking the menu icon or pressing  - . This allows you to view related information on zones, locations, stockpiles, workshops and farm plots by selecting the corresponding tabs.

This menu has replaced the **building list screen** from v0.47 of the game.
