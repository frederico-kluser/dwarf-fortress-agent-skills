# World gen.txt

> Fonte: [World gen.txt](https://dwarffortresswiki.org/index.php/World_gen.txt) — Dwarf Fortress Wiki (GFDL/MIT)

The **world_gen.txt** file stores presets for advanced world generation. If saved in-game, the file is created in the ```/prefs/` folder, and can be manually edited to use values that are outside the limits allowed by the in-game interface.
