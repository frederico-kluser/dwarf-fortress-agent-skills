# Alabaster

> Fonte: [Alabaster](https://dwarffortresswiki.org/index.php/Alabaster) — Dwarf Fortress Wiki (GFDL/MIT)

**Alabaster** is a fine-grained variety of gypsum. It is found within gypsum in small clusters, and can likewise be used to make plaster. There is no in-game difference between the two materials except in coloration.
