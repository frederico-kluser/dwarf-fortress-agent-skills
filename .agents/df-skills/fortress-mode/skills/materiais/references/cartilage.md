# Cartilage

> Fonte: [Cartilage](https://dwarffortresswiki.org/index.php/Cartilage) — Dwarf Fortress Wiki (GFDL/MIT)

**Cartilage** is a body part generated from butchering most animals. It has no use in the meat industry, and can be dumped without a second thought.

Some fish, such as sharks, are notable for having cartilaginous skeletons. These fish do not drop bones, only cartilage.

Note that dwarves consider cartilage useful (like bones), so they will not dump it automatically, even if you have the -dump other option enabled. It does not rot on its own either.
