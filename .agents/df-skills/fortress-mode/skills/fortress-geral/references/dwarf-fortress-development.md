# Dwarf Fortress Development

> Fonte: [Dwarf Fortress Development](https://dwarffortresswiki.org/index.php/Dwarf_Fortress_Development) — Dwarf Fortress Wiki (GFDL/MIT)

**Dwarf Fortress Development** is the name of the page which describes the intended short-term development of *Dwarf Fortress*, although in practice the page lists development which will take place over several years. It is sometimes referred to as "dev_next", as this was the name of the web page's HTML file before the website update. Currently, Dwarf Fortress Development in now located at "dev".
