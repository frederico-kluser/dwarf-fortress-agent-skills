# Dwarf cancels task: Hunting vermin for food

> Fonte: [Dwarf cancels task: Hunting vermin for food](https://dwarffortresswiki.org/index.php/Dwarf_cancels_task:_Hunting_vermin_for_food) — Dwarf Fortress Wiki (GFDL/MIT)

If you see this message, or a dwarf with the "Hunt for small creature" job, the dwarf can't find any food.

A dwarf may not be able to find any food because:

1.  There is no more food for any dwarf. (Did you bring enough on embark? Did you trade for enough from the traders? Did you cook all your edible seeds?)
2.  This dwarf is hungry, but is blocked from accessing the food stores. (Is a door locked? Is there flooding? Did an underground tree grow across a path? Did you mine out stairs or ramp you shouldn't have?)

See starvation for more details.
