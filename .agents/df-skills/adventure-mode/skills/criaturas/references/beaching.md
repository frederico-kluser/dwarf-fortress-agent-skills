# Beaching

> Fonte: [Beaching](https://dwarffortresswiki.org/index.php/Beaching) — Dwarf Fortress Wiki (GFDL/MIT)

\ **Beaching** is a characteristically odd behavior carried out by creatures with a creature token. These aquatic animals will occasionally strand themselves on land, where they are immobile and unable to breathe. This is intended to model the poorly-explained real-life behavior of similar animals; see cetacean stranding on Wikipedia for more information. \
