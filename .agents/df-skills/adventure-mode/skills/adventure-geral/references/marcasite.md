# Marcasite

> Fonte: [Marcasite](https://dwarffortresswiki.org/index.php/Marcasite) — Dwarf Fortress Wiki (GFDL/MIT)

**Marcasite** is an unremarkable white stone found within kaolinite. While it can be used as building material for floors, walls and such, it can not be used for furnaces and forges.

In real life, it is similar to pyrite but is much more brittle and thus not usable as a gem.
