**Fishery worker** is a category profession. It contains skills related to the catching, cleaning, and preparation of fish for consumption. A dwarf whose highest skill is in this category will be known as a Fishery Worker.

The fishery worker skills are:

|:--:|:--:|
|  | Fishery Worker |
|   | Fish cleaner / Fish dissector / Fisherdwarf |
