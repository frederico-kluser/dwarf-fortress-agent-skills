# Leader

> Fonte: [Leader](https://dwarffortresswiki.org/index.php/Leader) — Dwarf Fortress Wiki (GFDL/MIT)

*This page is about the Skill. For the Noble position, see Expedition leader.*

**Leader** is a Miscellaneous Skill. Toady One stated in a Fortress talk that it is used when a dwarf tries to teach another dwarf a skill through a lecture. It is currently only used to train weapon skills. All squad leaders (Militia commander, Militia captains, the Captain of the guard and the Hammerer) will learn it slowly.
