*This page is about the Skill. For the Noble position, see Expedition leader.*

**Leader** is a miscellaneous skill. Toady One stated in a Fortress talk that it is used when a dwarf tries to teach another dwarf a skill through a lecture.\[Verify\] It is currently only used to train weapon skills. All squad leaders (militia commander, militia captains, the captain of the guard and the hammerer) will learn it slowly.
