# Engineer

**Engineer** is a category profession. It contains skills relating to the construction and operation of mechanisms and machinery, including siege engines.

An adult dwarf whose best skill is in this category, and who has accumulated ranks in other engineering skills (exactly how much is not clear) will be known as an Engineer, or once they have achieved 'grand master' skill in one of these skills, but not in the others.

If a dwarf has experience in only one engineer skill, they will be known by their highest skill.

The engineering skills are:

|  |  |
|:--:|:--:|
|  | Engineer |
|   | Mechanic / Pump operator / Siege engineer / Siege operator |

The optics engineering and fluid engineering skills don't count as engineering skills, but contribute to the Engineer title regardless and are taught in the respective guildhall alongside Student skill.
