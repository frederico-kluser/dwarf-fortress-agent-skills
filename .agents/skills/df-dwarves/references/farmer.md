# Farmer

xTATTEREDx  · +FINE+  · \*SUPERIOR\*  · ≡EXCEPTIONAL≡  · ☼MASTERWORK☼

**Farmer** is a profession used for the production and use of plants, meat, plant fiber cloth, and beekeeping products. A dwarf whose highest skill is in this category is known as a Farmer. Graphically, those assigned to farmer jobs will be seen in brown clothing.

Confusingly, the skill associated with the Farming (Fields) labor is actually Planter.

The farming skills are:

|  |  |
|:--:|:--:|
|  | Farmer |
|   | Beekeeper / Brewer / Butcher / Cheese maker / Cook / Dyer / Gelder / Planter / Herbalist / Lye maker / Milker / Miller / Potash maker / Presser / Shearer / Soaper / Spinner / Tanner / Thresher / Wood burner |
