# Chasm

> Fonte: [Chasm](https://dwarffortresswiki.org/index.php/Chasm) — Dwarf Fortress Wiki (GFDL/MIT)

''This article is about the biome. You might be looking for a certain bottomless feature found here (spoilers!).

Bottomless **chasms** as a geographical feature do not exist. However, chasms exist as the SUBTERRANEAN_CHASM biome token in the raws. This tag seems to imply the spawning of creatures in non-water parts of the caverns, as opposed to the SUBTERRANEAN_WATER biome.

Bottomless chasm **tiles** can be placed in the Arena mode layout (arena.txt) using the character "C". When something falls into one of these tiles, the tile has an animation of (note that in-game with the default tileset, the first dot will appear larger). Open space above a chasm will always use the tile, never a tile.
