# Note

> Fonte: [Note](https://dwarffortresswiki.org/index.php/Note) — Dwarf Fortress Wiki (GFDL & MIT). Snapshot 2026-06.

!!UNKNOWN!!  · xTATTEREDx  · +FINE+  · \*SUPERIOR\*  · ≡EXCEPTIONAL≡  · ☼MASTERWORK☼

**Notes** were comments that you could leave around your fortress. Instead, now many elements of the game can be renamed:

- Names and titles of all creatures
- Squads
- All buildings, not including constructions
- Stockpiles
- Zones, locations, and burrows

Notes were removed in 0.50 and later versions of Dwarf Fortress, however this feature has been reinstated with a DFHack plugin.
