### Profession Categories

Layered graphics only use profession categories rather than individual professions. These are:

-  STONEWORKER
-  MINER
-  METALSMITH
-  ENGINEER
-  FARMER
-  WOODWORKER
-  JEWELER
-  RANGER
-  STANDARD (i.e. "Peasant". Also includes some invader & visitor professions.)
-  CRAFTSMAN
-  FISHERY_WORKER
-  MERCHANT
-  CHILD
-  NONE (applies to anything not included above - doctors, bards, military, etc. Note that this is distinct from having no profession condition; it will apply ONLY to NONE units.)

Noble positions do not affect the categorization; a craftsdwarf appointed to mayor will still use the layers designated for `CRAFTSMAN`.
