**Ammo tokens** are an item definition token for ammunition.

## Tokens

|  |  |  |
|----|----|----|
| Token | Arguments | Description |
|  NAME | singular:plural | What this item will be called in-game. |
|  CLASS | anything | This ammo can be fired from a weapon that is set to fire the same ammo type. Defaults to BOLT. |
|  SIZE /  WEIGHT | value | How large the ammunition is. Required. |
|  ATTACK | attacktype:BLUNT or EDGE / contact_area:value / penetration_size:value / verb2nd:string / verb3rd:string / noun:string / velocity_multiplier:value | Specifies the attack characteristics of the ammunition. |
