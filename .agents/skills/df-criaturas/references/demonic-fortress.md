# Demonic fortress

> Fonte: [Demonic fortress](https://dwarffortresswiki.org/index.php/Demonic_fortress) — Dwarf Fortress Wiki (GFDL/MIT)

In older versions of the game, demonic fortresses made of smooth slade could be found. These have been officially removed since the release of 0.40.01, but may still be added manually using a(n) (experimental) DFhack script.
