# Multiply value

> Fonte: [Multiply value](https://dwarffortresswiki.org/index.php/Multiply_value) — Dwarf Fortress Wiki (GFDL & MIT). Snapshot 2026-06.

xTATTEREDx  · +FINE+  · \*SUPERIOR\*  · ≡EXCEPTIONAL≡  · ☼MASTERWORK☼

A creature's [`[MULTIPLY_VALUE]`](/index.php/Material_definition_token#MULTIPLY_VALUE "Material definition token") affects the value of animal products such as meat, raw hides, silk thread, and anything made from these raw materials, including leather or silk clothing. While it's possible for [`[MULTIPLY_VALUE]`](/index.php/Material_definition_token#MULTIPLY_VALUE "Material definition token") to apply to only specific parts of a creature, all non-modded creatures use `SELECT_MATERIAL:ALL`, so all parts of the animal are multiplied.

The giant and animal person versions, if not listed, have the same [`[MULTIPLY_VALUE]`](/index.php/Material_definition_token#MULTIPLY_VALUE "Material definition token") as the base animal.
